本项目采用sqlite数据库游戏规则
🏆 胜利条件： 击杀对方的 “枭” 即可获胜。

♟️ 棋子功能
枭 (X - General): 全军核心，攻防（攻击和受击）均加成 1.3倍。
散 (S - Soldier): 普通步兵，用于肉搏。
矢 (T - Archer): 远程杀手！
距离 2 格：威力 120% (最佳射程)
距离 3 格：触发特殊狙击 (若破防则无伤击杀)
卡片功能（只影响决斗判定）
二定一: 二进制数的第二位是1，即数值一定大于16，每局共7张。
头定一: 二进制数的第一位为1，即数值一定大于32，每局共5张。
双星高照： 二进制数的前两位为1，即数值一定大于48，每局共3张。
三阳开泰：二进制数的前三位为1， 即数值一定大于56，每局共2张。
移动和攻击
随机数（核心规则）: 移动和工具的结果均由本随机数产生。随机数是6个二进制数组成，移动的结果为6个二进制数相加的和，攻击的结果为6个随机二进制数所转化成的10进制数
移动: 进行投彩后，根据步数移动任意棋子，每次移动均只能够移动1格。如果投彩结果为0需点击“跳过回合”按钮跳过本回合
攻击: 弹出决斗弹窗后，点击决斗按钮，决斗数值为随机数和棋子加成倍率的乘积，除枭外，所有棋子受击倍率均为1。
🗺️ 地图机制
根据选择的地图不同，场上会出现灰色墙壁和高台。墙壁不可逾越，不可射穿。登上高台后散棋无法攻击，矢棋有攻击距离和伤害值提高

启动命令推荐： gunicon
nohup python3 -m gunicorn --worker-class eventlet -w 1 -b 127.0.0.1:5211 app:app > output.log 2>&1 &
